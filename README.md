# AirBnB-Clone
Backend for Airbnb clone built using Node.js, Express.js, and MongoDB


This is a backend clone of Airbnb built using Node.js, Express.js, and MongoDB.
The project focuses on implementing core backend functionalities such as
authentication, property listings, bookings, and user management.

This project was created as part of my backend development learning journey.


# Features
- User Authentication (Signup & Login)
- JWT-based Authentication & Authorization
- Property Listing (Create, Update, Delete)
- Booking System
- MongoDB Database Integration using Mongoose
- RESTful API Architecture
- Secure Password Hashing using bcrypt
- Environment-based Configuration


#Tech-Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (JSON Web Token)
- bcrypt
